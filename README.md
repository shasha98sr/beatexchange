# BeatExchange

BeatExchange is a social platform for beatboxers to share their beats, get feedback, and connect with other artists in the community.

## Features

- Upload and share beatbox audio clips
- Play back audio with a waveform visualization
- Leave comments and feedback on beats
- User profiles and authentication
- Feed of recent uploads
- Like and favorite beats

## Tech Stack

- Backend: Python/Flask
- Frontend: React
- Database: PostgreSQL
- Audio Processing: Web Audio API
- Authentication: JWT

## Setup

1. Install backend dependencies:
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Install frontend dependencies:
```bash
cd frontend
npm install
```

3. Start the development servers:

Backend:
```bash
cd backend
flask run
```

Frontend:
```bash
cd frontend
npm start
```
